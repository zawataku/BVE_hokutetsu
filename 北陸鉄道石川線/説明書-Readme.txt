▼サンプル路線の切り替えについて
路線制作キットは初期状態ではサンプル路線を運転できるようになっています。
Scenarioフォルダ内のRouteKit.txtを編集し、
;Route = RouteKit\Map.txt
Route = RouteKit\MapSample.txt
となっているところを、
Route = RouteKit\Map.txt
;Route = RouteKit\MapSample.txt
とコメントアウトを付け替えることによって、地面・背景、自線と右に他線が存在する複線の線路が存在する、ほぼ空のマップファイルに切り替えることができます。制作される方はマップファイルの切り替え忘れに気を付けてください。

サンプル路線では左右違いを除いた全てのストラクチャを設置しているのでストラクチャ設置の参考にしてください。
特に駅プラットホームと分岐器についてはRouteKitフォルダ内に説明文書を同封してあります。


▼ストラクチャデータについて
キット同梱のストラクチャデータはBve trainsim5用に制作されたものです。
Bve trainsimアドオンデータの制作に際して、これらのストラクチャの利用は改造・無改造問わず自由です。
データ公開に際しての連絡も不要です。
路線制作キットが素敵な路線データ作りの一助になれば嬉しく思います。

なお、本キットのストラクチャデータを改造した上で、「単体で配布したい場合」には以下もお読みください(路線データに組み込んでの公開の場合には以下の添付文書は不要です)。
トラブル防止のため、
・データの利用条件
・二次配布者や改造を担当された方のHNとサイトURLやメールアドレス等の連絡先
・ストラクチャデータを改造した旨
を記載した添付文書(説明書、ReadMe等)を同梱の上で公開するようお願いします。

▼分岐器検討2019シートから生成した分岐器ストラクチャ類の扱いについて
分岐器検討2019シートから生成した分岐器ストラクチャの利用、改造、配布について、とまりの許可を得る必要はありません。ご自分で作られたストラクチャデータとして扱うようお願いします。
このシートは扱いやすさの考慮をしていません。不可解な点が多々あるとは思いますが、適宜改良して利用されるようお願いします。

▼キット同梱の車両データについて
保安措置プラグイン(Ats.dll)はBve5公式サイトのサンプルソースより作成されています。
このプラグインをBvetrainsimで使用する以外の目的で利用することを認めません。

キットに同梱された車両データを路線データに組み込んで配布することについて、とまりの許可を得る必要はありませんが、
トラブル防止のため、
・車両データの原作者がとまりである旨
・改造した場合には改造した旨
・路線データ制作担当者HNとサイトURL、連絡先
を記載した添付文書を配布データに同梱の上で公開するようお願いします。

▼キット同梱の車両データ中の一部のファイルを利用したい場合について
トラブル防止のため、
・利用したファイルの原作者がとまりである旨
・ファイルを改造した場合には改造した旨
を添付文書に追記した上で公開・配布するようお願いします。
